# Support Policy

## Compatibility

- PHP: `8.2`, `8.3`, `8.4`, `8.5`
- Laravel examples: `12`, `13` (app-layer transport example only)

## What is supported

- SDK API bugs
- Verification flow correctness
- Typed model mapping issues
- CI and static analysis quality regressions

## Response target

- Best effort for OSS support
- Security issues prioritized
- Bugfix releases as needed
